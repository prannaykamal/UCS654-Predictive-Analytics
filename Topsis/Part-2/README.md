# TOPSIS-Prannay-102317125

A Python package for implementing the Technique for Order of Preference by Similarity to Ideal Solution (TOPSIS). 
TOPSIS is a multi-criteria decision analysis method that chooses the alternative which is closest to the ideal solution and farthest from the negative-ideal solution.
